# Папка для фото портфолио

Сюда нужно положить реальные фото работ мастеров (с согласия клиентов на публикацию!),
с именами файлов, указанными в config.json → "portfolio".

Например:
- anna_1.jpg, anna_2.jpg — работы Анны
- balayage_1.jpg, balayage_2.jpg — примеры балаяжа

Бот сам подхватит их, когда клиент попросит показать примеры
("покажи балаяж", "есть фото работ Анны" и т.п.).

Если файла нет — бот вежливо скажет, что фото пока не загружены,
вместо того чтобы упасть с ошибкой.
